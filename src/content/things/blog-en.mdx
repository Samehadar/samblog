This very site — a multilingual personal blog where I write about engineering leadership, scaling teams, technical decisions, and lessons learned along the way.

## Tech stack

- **Astro 6** — static site generator with file-based routing
- **Vue 3** — interactive components (blog filter)
- **Tailwind CSS 4** — utility-first styling via CSS custom properties
- **MDX** — content authoring with component support
- **i18n** — manual routing for Russian, English, and Polish
- **RSS** — auto-generated feeds per language + global feed

Built from scratch, designed with inspiration from [Rands in Repose](https://randsinrepose.com/), [All Things Distributed](https://allthingsdistributed.com/), and [yegor256.com](https://www.yegor256.com/).
