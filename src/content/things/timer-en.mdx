I used an online Pomodoro timer for over 10 years — I always loved its simplicity. At some point I decided to build my own version from scratch, so it's always available and doesn't depend on third-party services.

## Features

- Three timer modes: Pomodoro (25m), Short Break (5m), Long Break (10m)
- Start / Stop / Reset with keyboard shortcuts
- Audio notifications with 5 sound options and volume control
- Desktop notifications via Notification API
- Session log with localStorage persistence
- Pomodoro goal tracker
- Fully customizable intervals
- Auto-start pomodoro/break sequences
