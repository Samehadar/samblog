## Возможности

- Три режима: Помодоро (25 мин), Короткий перерыв (5 мин), Длинный перерыв (10 мин)
- Старт / Стоп / Сброс с поддержкой горячих клавиш
- Звуковые уведомления с 5 вариантами звука и регулировкой громкости
- Уведомления на рабочем столе через Notification API
- Журнал сессий с сохранением в localStorage
- Трекер дневной цели по помидорам
- Полностью настраиваемые интервалы
- Автоматический запуск последовательностей помодоро/перерыв
