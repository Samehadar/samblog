Этот самый сайт — мультиязычный персональный блог, где я пишу про инженерное лидерство, масштабирование команд, технические решения и уроки, которые вынес по дороге.

## Технологический стек

- **Astro 6** — генератор статических сайтов с файловой маршрутизацией
- **Vue 3** — интерактивные компоненты (фильтр блога)
- **Tailwind CSS 4** — стилизация через CSS custom properties
- **MDX** — контент с поддержкой компонентов
- **i18n** — ручная маршрутизация для русского, английского и польского
- **RSS** — автогенерация фидов по языкам + глобальный фид

Создан с нуля, дизайн вдохновлён [Rands in Repose](https://randsinrepose.com/), [All Things Distributed](https://allthingsdistributed.com/) и [yegor256.com](https://www.yegor256.com/).
